Algunos lenguajes de la web

HTML + CSS + JAVASCRIPT

  • HTML5

    HTML5 (Hyper text markup lenguage, versión 5) es la quinta revision importante del lenguaje básicode la world wide web. HTML

    Usa la extensión .html

  • CSS

    Hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets) es un lenguaje usado para definir y crear la presentacion de un documento estructurado escrito en HTML.

    Usa la extensión .css

  • JS

    JavaScript (abreviado comúnmente "JS") es un lenguaje de programacioninterpretado utilizado principalmente en el lado del cliente (client-side),implementado como parte de un navegador web permitiendo la mejora de la interfaz de usuario y páginas web dinámicas

    Usa la extensión .js
